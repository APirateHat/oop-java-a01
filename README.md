# Compiling:

## Task 1:

To compile task 1 open the terminal. Find and change to the **task_01** directory.

Assuming you have Java installed you can type `javac *.java` to compile all scripts.
Afterwards excecute the main java file with `java Main`.


## Task 2

Task 2 is compiled with Gradle. Find and change to the **task_02** directory. 
In the terminal type `./gradlew run --console plain -q`

## Task 3
